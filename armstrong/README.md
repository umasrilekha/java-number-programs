# Armstrong Number Checker (Java)

## Description

This project checks whether a given number is an **Armstrong number** using Java. It demonstrates digit extraction and power calculations using loops.

## Problem Statement

Write a Java program to check if a number is an Armstrong number.

An Armstrong number is a number that is equal to the sum of its digits each raised to the power of the number of digits.

## Approach

1. Count the number of digits in the number
2. Extract each digit using modulus (`% 10`)
3. Raise each digit to the power of total digits
4. Add the results
5. Compare the sum with the original number
   
## Code Features

* Uses loops (for / while)
* Uses `Math.pow()` for exponent calculation
* Beginner-friendly implementation
* 
## Example

Input:
153
Output:
Armstrong Number
