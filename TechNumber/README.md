# Tech Number Checker (Java)

## Description

This project checks whether a given number is a **Tech Number** using Java. It focuses on splitting numbers and applying mathematical operations.

## Problem Statement

Write a Java program to check if a number is a Tech Number.

A Tech Number is a number with an even number of digits such that when the number is split into two halves and summed, the square of the sum equals the original number.

## Approach

1. Count the number of digits
2. Check if the number of digits is even
3. Split the number into two halves
4. Add both halves
5. Square the sum
6. Compare the result with the original number

## Code Features

* Uses loops and conditionals
* Uses arithmetic operations for splitting numbers
* Simple and clear implementation

## 🧪 Example
Input:
2025
Output:
Tech Number

Explanation:
20 + 25 = 45  
45² = 2025 ✅
